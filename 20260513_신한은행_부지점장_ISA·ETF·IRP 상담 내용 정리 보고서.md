# ISA·ETF·IRP 상담 내용 정리 보고서

## 1. 상담 전체 결론

이번 상담의 핵심은 다음입니다.

> 기존에 방치된 ISA·연금 계좌를 정리하고,
> 새 ISA 계좌를 만들어 **S&P500·나스닥 같은 미국 지수형 ETF/펀드**를 ISA 안에서 운용하면 세금 혜택을 더 잘 활용할 수 있다는 내용입니다.
> IRP와 기존 연금저축은 당장 해지하지 말고, 55세 이후 연금 수령 시점에 통합 관리하는 방향이 제안되었습니다. 

---

# 2. ISA 기본 구조

ISA는 **개인종합자산관리계좌**입니다.
쉽게 말하면, 여러 금융상품을 담을 수 있는 **절세용 바구니 계좌**입니다.

| 항목           | 내용                     |
| ------------ | ---------------------- |
| 계좌 성격        | 절세 계좌                  |
| 담을 수 있는 상품   | 예금, 펀드, ETF, ELS/ELB 등 |
| 연 납입한도       | **2,000만 원**           |
| 총 납입한도       | **5년간 최대 1억 원**        |
| 의무가입기간       | **3년**                 |
| 일반형 비과세      | 순수익 200만 원까지           |
| 서민형·농어민형 비과세 | **순수익 400만 원까지**       |
| 초과수익 과세      | **9.9% 분리과세**          |

현재 ISA는 연 2,000만 원, 총 1억 원까지 납입 가능하고, 의무가입기간 3년을 채워야 세제혜택을 받을 수 있습니다. 일반형은 순이익 200만 원, 서민형·농어민형은 400만 원까지 비과세이고 초과분은 9.9% 분리과세로 안내됩니다. ([나무증권][1])

# 3. “1억은 몇 년?” 정리

**1억 원은 5년 납입 기준**입니다.

| 연도  |   납입 가능액 |
| --- | -------: |
| 1년차 | 2,000만 원 |
| 2년차 | 2,000만 원 |
| 3년차 | 2,000만 원 |
| 4년차 | 2,000만 원 |
| 5년차 | 2,000만 원 |
| 합계  | **1억 원** |

정리하면:

> **3년은 의무가입기간**,
> **5년은 최대 1억 원 납입 기준**입니다.

즉, 3년만 지나도 세제혜택 요건은 충족할 수 있지만, 납입한도를 최대한 채우면 5년 동안 1억 원까지 넣을 수 있다는 뜻입니다.

---

# 4. 400만 원 비과세의 의미

상담 중에 들은 **“400만 원까지 비과세”**는 맞습니다. 

다만 여기서 400만 원은 **넣은 돈 400만 원**이 아니라, ISA 안에서 발생한 **순수익 400만 원**을 말합니다.

예를 들어 서민형 또는 농어민형 ISA라고 가정하면:

|   ISA 순수익 | 세금                                |
| --------: | --------------------------------- |
| 300만 원 수익 | 전액 비과세                            |
| 400만 원 수익 | 전액 비과세                            |
| 600만 원 수익 | 400만 원 비과세 + 초과 200만 원은 9.9% 분리과세 |

즉:

> **ISA 안에서 벌어들인 이자·배당·ETF 수익 등을 합산한 순이익 중 400만 원까지 세금을 안 낸다**는 의미입니다.

# 5. 일반 금융소득 세금 15.4%

일반적으로 예금 이자나 배당금에는 **15.4% 세금**이 붙습니다.

| 구분          |        세율 |
| ----------- | --------: |
| 이자소득세/배당소득세 |       14% |
| 지방소득세       |      1.4% |
| 합계          | **15.4%** |

예를 들어 예금 이자가 100만 원이면:

| 항목       |         금액 |
| -------- | ---------: |
| 이자       | 1,000,000원 |
| 세금 15.4% |   154,000원 |
| 실제 수령액   |   846,000원 |

그런데 ISA 안에서는 일반형 200만 원, 서민형·농어민형 400만 원까지 비과세되고, 초과분도 9.9% 분리과세라서 일반계좌보다 유리할 수 있습니다.

# 6. 국내주식은 왜 “세금이 없다”고 했나

상담자가 말한 **“국내주식은 세금이 없다”**는 말은 정확히는 **국내 상장주식의 매매차익에 대해 일반 개인투자자는 보통 양도소득세를 내지 않는다**는 뜻입니다. 

| 구분              | 세금                   |
| --------------- | -------------------- |
| 국내 상장주식 매매차익    | 일반 소액주주는 보통 양도소득세 없음 |
| 국내 주식형 ETF 매매차익 | 보통 비과세               |
| 국내 주식 배당금       | 15.4% 배당소득세          |
| 주식 매도 시         | 증권거래세 있음             |
| 대주주             | 양도소득세 대상             |

국세청 안내 기준으로 상장주식 대주주는 코스피 1%, 코스닥 2%, 코넥스 4% 이상 또는 시가총액 50억 원 이상 등의 요건에 해당하면 양도소득세 신고 대상입니다. 일반적인 소액투자자의 장내거래는 여기에 해당하지 않는 경우가 많습니다. ([NTS][2])

따라서 상담자의 말은:

> 국내주식 매매차익은 원래 세금이 거의 없으니,
> ISA에 국내주식만 담는 것은 절세효과가 상대적으로 작다.

는 의미입니다.

# 7. 왜 S&P500·나스닥을 추천했나

상담자가 S&P500·나스닥을 말한 이유는, ISA의 세제혜택을 더 잘 활용할 수 있기 때문입니다. 

S&P500과 나스닥은 미국 주식시장 지수입니다.

| 구분       | 의미                   |
| -------- | -------------------- |
| S&P500   | 미국 대표 500개 기업 지수     |
| 나스닥100   | 미국 기술주 중심 100개 기업 지수 |
| 투자 방식    | ETF 또는 펀드로 투자        |
| ISA와의 관계 | ISA 안에 담아 운용 가능      |

상담자가 말한 “S&P500이나 나스닥에 투자한다”는 것은 보통 미국 주식을 하나씩 직접 사는 것이 아니라, **S&P500 ETF, 나스닥100 ETF, 또는 해당 지수를 추종하는 해외펀드에 투자한다**는 뜻입니다.

# 8. ETF란 무엇인가

ETF는 **상장지수펀드**입니다.
쉽게 말하면 **주식처럼 사고파는 펀드**입니다.

예를 들어 S&P500 ETF를 산다는 것은:

> 애플, 마이크로소프트, 엔비디아 등 미국 대표 기업 500개에 나눠 투자하는 상품을 사는 것

과 비슷합니다.

| 투자 방식      | 의미                       |
| ---------- | ------------------------ |
| 삼성전자 주식    | 삼성전자 한 회사에 투자            |
| S&P500 ETF | 미국 대표 500개 기업에 분산 투자     |
| 나스닥100 ETF | 미국 기술주 중심 100개 기업에 분산 투자 |

ETF는 예금이 아니므로 원금보장이 되지 않습니다.

| 시장 상황  | ETF 결과             |
| ------ | ------------------ |
| 지수가 오름 | ETF 가격도 대체로 상승, 수익 |
| 지수가 내림 | ETF 가격도 대체로 하락, 손실 |

---

# 9. ETF 수익 실현 구조

ETF는 **10% 오르면 자동으로 돈이 들어오는 상품이 아닙니다.**

기본 구조는 이렇습니다.

| 단계 | 내용                     |
| -- | ---------------------- |
| 1  | ISA 안에서 ETF를 매수        |
| 2  | ETF가 10% 상승            |
| 3  | 평가수익 발생                |
| 4  | 내가 매도하거나 은행에 매도 지시     |
| 5  | 매도 후 ISA 계좌 안 현금으로 들어옴 |

예를 들어 1,000만 원을 투자했을 때:

| 상황     |       평가금액 |
| ------ | ---------: |
| 10% 상승 |   1,100만 원 |
| 매도 전   |    아직 평가수익 |
| 매도 후   | 수익 확정, 현금화 |

즉:

> **10% 수익이 났을 때 팔아야 실제 수익이 확정됩니다.**

자동매도를 하려면 증권사 앱이나 상품 구조에서 **예약매도, 조건부매도, 자동감시주문** 같은 기능이 가능한지 확인해야 합니다.

# 10. 계속 내리면 어떻게 되나

ETF는 예금처럼 만기가 되어 원금이 반환되는 구조가 아닙니다.

| 상황      | 결과                |
| ------- | ----------------- |
| ETF가 오름 | 평가수익              |
| ETF가 내림 | 평가손실              |
| 기간이 지남  | 자동 반환 없음          |
| 내가 매도함  | 그 시점의 수익 또는 손실 확정 |
| 계속 보유   | 계속 투자 상태 유지       |

즉, “10% 될 때까지 기다리는 구조”는 상품 자체의 약속이 아니라 **본인이 정한 투자전략**입니다.

예를 들면:

| 전략       | 설명               |
| -------- | ---------------- |
| 목표수익률 전략 | 10% 오르면 매도       |
| 장기보유 전략  | 3년, 5년, 10년 보유   |
| 분할매수 전략  | 한 번에 사지 않고 나눠 매수 |
| 손절 전략    | 일정 손실이면 매도       |

상담자가 말한 ISA 내 S&P500·나스닥 투자는 보통 **단기 10% 수익 실현**보다는 **3년 이상 장기 운용하면서 세제혜택을 받는 방향**에 가깝습니다.

# 11. 인버스 ETF 개념

인버스 ETF는 일반 ETF와 반대로 움직이는 상품입니다.

| 시장 움직임 | 일반 ETF | 인버스 ETF |
| ------ | -----: | ------: |
| 지수 상승  |     수익 |      손실 |
| 지수 하락  |     손실 |      수익 |

즉, “내리면 수익 나는 상품”은 일반 ETF가 아니라 **인버스 ETF**입니다.

다만 인버스 ETF는 보통 단기 대응용입니다. 장기투자용으로는 위험이 큽니다. 상담자가 말한 S&P500·나스닥 투자는 일반적으로 **인버스가 아니라 일반 지수형 ETF/펀드**를 의미한 것으로 보는 것이 자연스럽습니다.

# 12. 해외 ETF/해외펀드 과세 기준

해외 투자상품은 크게 3가지로 나눠야 합니다.

## 12-1. 국내주식형 ETF

예: KODEX 200, TIGER 200 등

| 구분   | 세금          |
| ---- | ----------- |
| 매매차익 | 보통 비과세      |
| 분배금  | 15.4% 배당소득세 |

## 12-2. 국내상장 해외 ETF

예: TIGER 미국S&P500, KODEX 미국나스닥100, ACE 미국S&P500 등

| 구분         | 세금                        |
| ---------- | ------------------------- |
| 매매차익       | 15.4% 배당소득세               |
| 분배금        | 15.4% 배당소득세               |
| ISA 안에서 운용 | 비과세 한도 적용 후 초과분 9.9% 분리과세 |

국내에 상장된 해외주식형 ETF는 매매차익에도 배당소득세 15.4%가 과세될 수 있습니다. 반면 국내주식형 ETF의 매매차익은 보통 비과세입니다. ([KB Capital][3])

## 12-3. 해외상장 ETF

예: 미국시장에 직접 상장된 VOO, SPY, QQQ 등

| 구분   | 세금                      |
| ---- | ----------------------- |
| 매매차익 | 연 250만 원 공제 후 22% 양도소득세 |
| 배당금  | 배당소득 과세                 |
| 신고   | 다음 해 5월 신고·납부 필요        |

해외상장 ETF는 해외주식처럼 취급되어 250만 원 초과 수익에 대해 22% 양도소득세가 적용됩니다. ([KB Capital][3])

# 13. ISA에서 S&P500·나스닥을 운용한다는 뜻

상담자가 말한 구조는 대략 이렇습니다.

> ISA 계좌를 만들고,
> 그 안에 S&P500 또는 나스닥100을 추종하는 ETF나 펀드를 담아서 운용한다.

즉:

| 구분            | 역할                     |
| ------------- | ---------------------- |
| ISA           | 절세 계좌, 바구니             |
| S&P500 ETF/펀드 | 실제 투자상품                |
| 나스닥100 ETF/펀드 | 실제 투자상품                |
| 은행/증권사        | 계좌 개설, 상품 추천, 운용 방식 제공 |

상담에서 말한 “S&P500, 나스닥 투자”는 미국 개별주식을 직접 사는 것이 아니라, **ISA 안에서 국내상장 미국지수 ETF 또는 해외펀드를 운용하는 방식**일 가능성이 큽니다.

# 14. 은행에서 “관리해준다”는 의미

상담자가 “관리해드릴게요”라고 한 표현은 반드시 일임형이라는 뜻은 아닙니다. 

ISA는 운용 방식에 따라 나뉩니다.

| 구분      | 누가 운용하나      | 특징                     |
| ------- | ------------ | ---------------------- |
| 중개형 ISA | 본인           | 증권사 앱에서 ETF/주식 직접 매매   |
| 신탁형 ISA | 본인이 상품 선택·지시 | 은행이 상담하고, 고객 지시에 따라 운용 |
| 일임형 ISA | 금융회사         | 투자성향에 맞춰 포트폴리오 운용      |

KB국민은행 안내에 따르면, 일임형 ISA는 고객이 전문가에게 자산운용을 일임하는 방식이고, 신탁형 ISA는 고객이 직접 펀드·ETF·ELS·예금 등을 선택하여 은행에 운용지시하는 방식입니다. ([KB 스타뱅크][4])

따라서 “관리해준다”는 말은 두 가지 가능성이 있습니다.

| 가능성    | 의미                                 |
| ------ | ---------------------------------- |
| 신탁형 관리 | 직원이 상품 추천·변경 상담을 해주지만 최종 지시는 본인이 함 |
| 일임형 관리 | 금융회사가 포트폴리오를 맡아서 운용함               |

은행에 반드시 확인해야 할 질문은 이것입니다.

> “제가 가입하는 ISA는 신탁형인가요, 일임형인가요?”
> “S&P500·나스닥 상품은 ETF인가요, 펀드인가요?”
> “매수·매도는 제가 직접 지시해야 하나요?”
> “10% 수익이 나면 자동 매도되는 구조가 있나요?”
> “원금손실 가능성이 있나요?”
> “수수료는 얼마인가요?”

# 15. IRP 상담 내용

사용자는 기업은행 IRP에 매년 700만~900만 원 정도 넣어 세액공제를 받아왔습니다. 상담자는 IRP를 **55세 이후 연금으로 나눠 받는 장기상품**이라고 설명했습니다. 

| 항목     | 내용                       |
| ------ | ------------------------ |
| 기존 IRP | 기업은행에 있음                 |
| 납입 방식  | 연말에 700만~900만 원 일시 납입    |
| 목적     | 세액공제                     |
| 운용     | 주로 채권·안정형                |
| 상담자 의견 | 장기상품이므로 단기 등락보다 장기 운용 필요 |
| 제안     | 신한 IRP를 새로 만들고 향후 관리     |

상담자는 기존 기업은행 IRP를 당장 해지하자는 취지는 아니고, 앞으로 신한 쪽에서 새 IRP를 만들고 관리하자는 방향으로 설명한 것으로 보입니다.

---

# 16. 기존 연금저축·연금신탁 내용

녹취상 농협생명 또는 농협 쪽에 오래된 연금저축·연금신탁이 조회된 것으로 보입니다. 

상담자는 다음과 같이 설명했습니다.

| 항목           | 내용                              |
| ------------ | ------------------------------- |
| 기존 연금저축/신탁   | 오래전에 가입된 상품 있음                  |
| 현재 상태        | 일부는 납입 중단 또는 소액 유지로 보임          |
| 연금 관련 총 납입한도 | 연 1,800만 원이라고 설명                |
| 상담 방향        | 기존 것은 무리하게 해지하지 말고 55세 이후 통합 검토 |
| 목적           | 나중에 연금수령 플랜을 하나로 짜기             |

즉, 기존 연금상품은 당장 없애는 것보다, 나중에 55세 이후 수령 시점에 합쳐서 관리하는 방향이 제안되었습니다.

# 17. 현재 자금 상황 정리

녹취상 사용자는 여유자금이 약 **2,000만~5,000만 원** 정도 있다고 말했습니다. 그중 약 **3,000만 원은 하나은행 1년 정기예금**에 들어가 있다고 했습니다. 

| 구분      | 내용                    |
| ------- | --------------------- |
| 여유자금    | 약 2,000만~5,000만 원     |
| 하나은행 예금 | 약 3,000만 원, 1년 정기예금   |
| 기존 ISA  | NH/나무 쪽에 있으나 잔액 거의 없음 |
| 기존 IRP  | 기업은행                  |
| 기존 연금   | 농협/농협생명 쪽 오래된 상품 존재   |

상담자의 방향은:

> 기존 NH/나무 ISA를 정리하고,
> 신한 ISA를 새로 만들어 2,000만 원부터 운용해 보자는 것

으로 보입니다.

# 19. 은행에 물어볼 질문 목록

은행에 다시 가면 아래 문장을 그대로 물어보면 됩니다.

## ISA 관련

> 제가 가입하려는 ISA는 **신탁형인가요, 일임형인가요?**

> S&P500이나 나스닥 투자는 **ETF로 하는 건가요, 펀드로 하는 건가요?**

> 제가 직접 매도 지시를 해야 하나요, 아니면 은행에서 포트폴리오를 알아서 조정하나요?

> 10% 수익이 나면 자동으로 매도하는 설정이 가능한가요?

> 이 상품은 원금손실 가능성이 있나요?

> 총 수수료는 얼마인가요?

## 세금 관련

> 저는 서민형 또는 농어민형 ISA 400만 원 비과세 대상인가요?

> ISA 안에서 발생한 수익이 400만 원을 넘으면 9.9% 분리과세가 맞나요?

## 기존 계좌 관련

> 기존 NH/나무 ISA를 해지해도 불이익이 없나요?

> 기업은행 IRP는 지금 이전하는 게 좋은가요, 55세 이후 통합하는 게 좋은가요?

> 농협 연금저축/연금신탁은 유지하는 게 좋은가요?

# 20. 최종 요약

이번 상담의 핵심은 다음 한 문장으로 정리할 수 있습니다.

> **ISA는 3년 이상 유지하면 세제혜택을 받을 수 있는 절세 계좌이고, 연 2,000만 원씩 5년간 최대 1억 원까지 납입할 수 있다. 사용자는 소득 조건상 서민형/농어민형에 해당할 가능성이 있어 순수익 400만 원까지 비과세 혜택을 받을 수 있다는 설명을 들었다. 국내주식 매매차익은 일반적으로 세금이 거의 없으므로 ISA 절세효과가 작고, 국내상장 S&P500·나스닥 ETF/펀드처럼 일반계좌에서는 15.4% 과세될 수 있는 해외지수형 상품을 ISA 안에서 운용하는 것이 세제상 유리하다는 상담이었다.**

다만 중요한 점은 이것입니다.

> **ETF는 예금이 아니고, 10% 오르면 자동으로 돈이 들어오는 상품도 아닙니다.**
> 수익이 나면 직접 매도하거나 은행에 매도 지시를 해야 수익이 확정됩니다.
> 은행이 “관리해준다”는 말이 일임형 운용인지, 단순 상담관리인지 반드시 확인해야 합니다.
